# libre

A simple website for experimenting with clojurescript and canvas

## Usage

```bash
lein deps
lein run
lein cljsbuild auto
```

Then point your browser at:

    http://localhost:8080/sketchbook
	
You can add sketches as you like, see `src/cljs/libre/sketches` for examples
